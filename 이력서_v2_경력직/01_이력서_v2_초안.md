## 대표 성과.

### 1. Node.js에 Spring `@Transactional`을 자체 구현 — 트랜잭션 전파 + Lazy Proxy

JavaScript에 없는 **트랜잭션 컨텍스트 전파**를 Proxy/Reflect 메타프로그래밍으로 구현. REQUIRED/REQUIRES_NEW/SUPPORTS 전파 정책을 지원하되, 실제 접근 시점에만 Bean을 clone하는 Lazy 전략으로 오버헤드를 최소화. 리팩토링이 부른 격리 회귀는 PostgreSQL `xmin` 기반으로 독립 트랜잭션을 검증해 해결. `@orkis/core`

### 2. 단일 코드베이스로 Cloud · Desktop 동시 지원

TS `paths` fallback과 `tsc-alias`를 조합해 **같은 import가 빌드 타깃에 따라 desktop override 우선 / main fallback**으로 자동 해석되게 설계. "exclude가 transitive import를 막지 못한다"는 비자명한 컴파일러 동작을 진단해 폴더 단위 cascade exclude로 해결. 클라우드 개발자 IDE에 desktop 코드가 노출되지 않게 분리. `orkis-backend`

### 3. 연구소 Kubernetes 플랫폼 GitOps 단독 구축

KT Cloud ManagedKS 위에 클러스터·네트워킹·스토리지·CI/CD·백업 전체를 매니페스트로 코드화. 노드풀 5종 분리, ingress-nginx DaemonSet+hostNetwork, cert-manager TLS 자동화, GitLab·Jenkins 자체 호스팅, 토큰 비노출 미러 백업 DR까지 인프라 전반을 단독 운영. `k8s-config`
